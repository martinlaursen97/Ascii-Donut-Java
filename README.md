# Ascii-Donut-Java

Controls:

Hold / Press
1 : rotate around x-axis.  
2 : rotate around y-axis.  
3 : rotate around z-axis.  
W : move light up.  
A : move light left.  
S : move light down.  
D : move light right.  
Q : move light towards camera.  
E : move light away from camera.  
R : reverse rotation.  
Space : stop rotation.  
Scroll wheel : Zoom in/out.
